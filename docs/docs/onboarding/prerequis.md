## Introduction

## Pré-requis

* [npm](https://www.npmjs.com/) version 9
* [nvm](https://github.com/nvm-sh/nvm) recommandé
* [NodeJS](https://nodejs.org/fr/), version 18
* [Docker](https://docs.docker.com/desktop/)
* [docker-compose](https://docs.docker.com/compose/)
